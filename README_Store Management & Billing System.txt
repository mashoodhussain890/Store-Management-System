# Store Management & Billing System

A simple C++ program for managing a store's inventory and generating customer bills. It uses text files to save product data and receipts.

## Features

**Manager Menu (Password: `admin123`)**
* Add, remove, and update products.
* View all available items.
* Save inventory to a text file (`products.txt`).

**Customer Menu**
* View the product list.
* Add items to a shopping cart.
* Generate a final bill with automatic age-based discounts.
* Save receipts to a text file (`bill.txt`).
